# Security Policy

## Reporting a vulnerability

Please report security issues privately.

- Do **not** open a public GitHub issue for vulnerabilities.
- Contact the maintainer directly and include:
  - A clear description of the issue
  - Steps to reproduce
  - Potential impact
  - Any suggested fix or mitigation

If you are unsure whether something is security-sensitive, report it privately first.

## Scope

This policy applies to this repository, including:

- `whisper-dictate.sh`
- `whisper-dictate.nix`

## Response expectations

- Initial acknowledgement target: within 72 hours
- Status updates: as fixes are investigated
- Coordinated disclosure: after a fix is available

Thanks for helping keep users safe.
